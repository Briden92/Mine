# Calculatrice Moderne

Une calculatrice web élégante et fonctionnelle avec une interface utilisateur moderne et responsive.

## Fonctionnalités

### 🔢 Opérations de base
- Addition (+)
- Soustraction (-)
- Multiplication (×)
- Division (÷)
- Modulo (%)

### 🎨 Interface moderne
- Design élégant avec gradient de fond
- Animations fluides
- Effets de survol et de clic
- Interface responsive (mobile et desktop)
- Thème sombre pour l'écran

### ⌨️ Support clavier complet
- **Nombres**: 0-9
- **Opérations**: +, -, *, /, %
- **Actions**:
  - `Enter` ou `=` : Calculer le résultat
  - `Backspace` : Effacer le dernier caractère
  - `Escape` : Effacement total (AC)
  - `.` : Point décimal

### ✨ Fonctionnalités avancées
- Formatage des nombres avec séparateurs de milliers
- Gestion des erreurs (division par zéro)
- Animation des boutons lors des pressions
- Easter egg caché (Konami Code)

## Installation et utilisation

### Option 1: Ouverture directe
1. Ouvrez simplement le fichier `index.html` dans votre navigateur web

### Option 2: Serveur local (recommandé)
```bash
# Démarrer un serveur HTTP simple
python -m http.server 8000
# ou
python3 -m http.server 8000
```
Puis ouvrez `http://localhost:8000` dans votre navigateur.

## Utilisation

### Interface graphique
- Cliquez sur les boutons pour saisir des nombres et opérations
- Le bouton `AC` efface tout
- Le bouton `⌫` efface le dernier caractère
- Le bouton `=` calcule le résultat

### Raccourcis clavier
- Utilisez votre clavier numérique ou les touches principales
- `Escape` pour tout effacer
- `Enter` pour calculer
- `Backspace` pour effacer

### Easter Egg
Tapez le Konami Code (↑↑↓↓←→←→BA) pour une surprise colorée! 🌈
